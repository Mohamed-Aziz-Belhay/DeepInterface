#  Web Deep Learning Model Tester

Ce projet est une application web permettant de **tester plusieurs modèles de Deep Learning en ligne**, directement depuis une interface simple et intuitive.  
L’utilisateur peut sélectionner un modèle, importer une image, lancer une prédiction et visualiser les résultats instantanément.

Ce projet est idéal pour :
- tester différents modèles (ResNet, MobileNet, modèles personnalisés…),
- mettre en valeur un travail de Deep Learning dans un rapport,
- exposer un backend d’inférence via une API.

---

##  Fonctionnalités

 Sélection d’un modèle pré-entraîné  
 Upload d’images via l’interface web  
 Affichage des prédictions (classe + score)  
 Interface simple, responsive et rapide  
 Backend API (FastAPI ou Flask)  
 Compatible avec PyTorch / TensorFlow  
 Extensible : ajout facile de nouveaux modèles
